# Star Stuff Counseling PLLC

> *"The cosmos is within us. We are made of star-stuff. We are a way for the universe to know itself."*
> — Carl Sagan

This repository contains the source code for the **Star Stuff Counseling PLLC** website — a static HTML/CSS/JavaScript site for a therapy practice based in Austin, TX.

---

## About the Practice

Star Stuff Counseling PLLC is founded and led by **Amie Parsons, MS LPC-A NCC ASDCS**, a Licensed Professional Counselor Associate supervised by Jennifer Gamewell, LPC-S. The practice helps clients find their North Star and foster meaningful connections with themselves and others.

Amie is a neurodiverse individual with backgrounds in education and special education, bringing a unique, compassionate perspective to therapy.

**Areas of Focus:**
- Complex Trauma (childhood, adolescence, sexual/physical/emotional abuse, financial struggles)
- Adoption Trauma & Attachment Issues
- Religious Trauma
- Neurodiversity (ADHD, Autism, Dyslexia, Dyscalculia, Learning Differences)
- LGBTQIA+ Affirming Care
- 504/IEP/Disability Navigation & Medical Gaslighting
- Anxiety, Depression & Life Transitions

---

## Therapy Modalities

- **EMDR** (Eye Movement Desensitization and Reprocessing)

## Site Structure

| File | Description |

---

## Technologies

- **HTML5** — page structure and content
- **CSS3** — custom styling (see `styles.css`)
- **Vanilla JavaScript** — responsive nav toggle and accordion UI (see `script.js`)
- **EmailJS** — contact form email delivery
- **Google Fonts** — Alice and Playfair Display typefaces
- **Font Awesome** — icons
- **OpenDyslexic3** font included for accessibility

---

## Running Locally

No build process is required. Simply open any `.html` file in a browser, or serve the directory with any static file server:

```bash
# Using Python 3
python3 -m http.server 8080
```

Then open `http://localhost:8080` in your browser.
